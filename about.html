About Agentry

Building the Future of Autonomous Communication

We're creating the infrastructure that will power the agentic internet

Our Story

How we're building the infrastructure for the agentic internet.

The Problem We Saw

As AI agents and autonomous systems became more prevalent, we noticed a critical gap: there was no standardized, reliable way for these systems to communicate with each other across organizational boundaries.

Existing solutions like email (SMTP) were designed for humans, while API brokers like Kafka required centralized infrastructure. Neither was suitable for the unique needs of autonomous systems.

Our Solution

We created AMTP (Agent Message Transfer Protocol) — a federated, asynchronous protocol specifically designed for agent-to-agent communication, with built-in security, reliability, and schema validation.

Combined with Agentry Gateway, our open-source implementation, this creates the foundation for a truly decentralized agentic internet.

Technology Focus

The technical principles that drive our development.

Open Source First

All core technology is open source under Apache 2.0 license, ensuring transparency and community-driven development.

Security by Design

Built-in digital signatures, encryption, and authentication ensure secure communication from day one.

High Performance

Designed for scale with asynchronous processing, efficient routing, and minimal overhead.

Schema-Aware

Structured JSON messages with schema validation enable type-safe communication and automatic API generation.
