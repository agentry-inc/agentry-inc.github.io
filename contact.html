Frequently Asked Questions

Quick answers to common questions about Agentry and AMTP.

What is AMTP and how is it different from existing protocols?
AMTP (Agent Message Transfer Protocol) is a federated, asynchronous protocol specifically designed for agent-to-agent communication. Unlike email (SMTP) which is human-centric, or API brokers like Kafka which require centralized infrastructure, AMTP provides structured, schema-aware messaging with built-in security and reliability features tailored for autonomous systems.
Is Agentry Gateway really open source?
Yes! Agentry Gateway is 100% open source under the Apache 2.0 license. You can view, modify, and distribute the code freely. Our commercial offerings (Agentry Cloud and Enterprise Solutions) provide managed infrastructure and additional features on top of the open-source foundation.
How do I get started with AMTP?
The easiest way to get started is to deploy our open-source Agentry Gateway using Docker. Check out our GitHub repository for quick start guides, sample configurations, and documentation. For managed infrastructure, you can start a free trial of Agentry Cloud.
What kind of support do you offer?
We offer community support through GitHub issues and discussions for open-source users. Agentry Cloud customers get email support with SLA guarantees. Enterprise customers receive dedicated support, including phone support, custom integrations, and 24/7 monitoring.
